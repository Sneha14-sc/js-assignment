Difference between var, let and const

var

'var' declarations are globally scoped or function/locally scoped. The scope is global when a 'var' variable is declared outside the function.This means that any variable i.e declared outside a function block is available for use in whole window. 'var' is function scoped when it is declared within a function. This means that it is available and can be accessed only within that function.

let

'let' is now preferred for variable declaration. It's a replacement to var declaration. A 'let' is a chunk of code bounded by {}. So the variable declared in the block with 'let' is only available for use within that block.

const

Variables declared within 'const' maintain constant values. 'const' declarations has some similarities with 'let' declarations. Like 'let' declarations, 'const' declarations can only be accessed within the block they were declared. The value of 'const' annot be updated or redeclared.
