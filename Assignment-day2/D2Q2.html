Array Methods

toString():The JavaScript method toString() converts an array to a string of (comma separated) array values.

join(): join() method also joins all array elements into a string.It behaves just like toString(), but in addition you can specify the separator

pop():The pop() method removes the last element from an array.

push():The push() method adds a new element to an array (at the end).

shift():The shift() method removes the first array element and "shifts" all other elements to a lower index.

unshift():The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements.

splice():The splice() method can be used to add new items to an array.

concat():The concat() method creates a new array by merging (concatenating) existing arrays:The concat() method does not change the existing arrays. It always returns a new array.

String Methods

indexOf():The indexOf() method returns the index of (the position of) the first occurrence of a specified text in a string. It cannot take powerful search values (regular expressions).

lastIndexOf():The lastIndexOf() method returns the index of the last occurrence of a specified text in a string. It searches backwards (from the end to the beginning).

search():b>The search() method searches a string for a specified value and returns the position of the match.It cannot take a second start position argument.

slice():It extracts a part of a string and returns the extracted part in a new string.var res = str.slice(7, 13);

substring():The substring() method is similar to slice().The difference is that substring() cannot accept negative indexes.var res = str.substring(7, 13);

substr():The substr() method is similar to slice().The difference is that the second parameter specifies the length of the extracted part.

replace():The replace() method replaces a specified value with another value in a string:tr.replace("Microsoft", "W3Schools");

trim();The trim() method removes whitespace from both sides of a string.

charAt():The charAt() method returns the character at a specified index (position) in a string.

concat():The concat() method joins two or more strings: text3 = text1.concat(" ", text2);

split():A string can be converted to an array with the split() method.
